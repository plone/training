========
Glossary
========

.. glossary::

   AWS
       `Amazon Web Services <https://aws.amazon.com/>`_ offers reliable, scalable, and inexpensive cloud computing services.
      Free to join, pay only for what you use.

   Linode
      `Linode.com <https://linode.com/>`_ is an American privately owned virtual private server provider company
       based in Galloway, New Jersey, United States.

   DigitalOcean
      `DigitalOcean, Inc. <https://www.digitalocean.com/>`_ is an American cloud infrastructure provider
       headquartered in New York City with data centers worldwide.
